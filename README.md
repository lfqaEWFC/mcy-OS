# mcy-OS

简体中文说明（Chinese）

## 项目简介

`mcy-OS` 是一个教学/实验性质的操作系统内核工程，包含从引导、内核、设备驱动、文件系统到用户进程的基本实现。代码以 C 与汇编混合实现，适合作为操作系统课程练习和学习参考。

## 代码结构概览

- `boot/`：引导程序与启动相关汇编代码。
- `kernel/`：内核入口、初始化、主循环及中断处理。
- `fs/`：文件系统实现（inode、目录、文件、超级块等）。
- `device/`：设备驱动（IDE、键盘、计时器、控制台等）。
- `lib/`：基础库，字符串、内存、IO 等工具与内核态 stdio 支持。
- `thread/`：线程/调度与同步实现。
- `user/`：用户态相关的系统调用与断言支持。
- `userprog/`：用户进程管理（exec、fork、wait/exit 等）。
- `shell/`：一个简单的命令行 shell 与内建命令实现。
- `command/`：一些用户命令示例（如 `cat`）。

（工程中还包含若干测试/演示程序与工具脚本）

## 快速开始（构建与运行）

先确保系统已安装必要工具：`gcc 4.4`、`make`、`nasm`、`bochs 2.6.8`，按需安装。

在项目根目录执行：

```bash
make all
```

本仓库自带 makefile 文件，需要在gcc4.4编译器的环境下进行 make 操作

在command目录下执行

```bash
bash -x compile.sh
```

最后在您安装的 bochs 目录下运行
运行操作详见：
[text](https://blog.csdn.net/kanshanxd/article/details/130689471?spm=1001.2014.3001.5506)

注：上述命令根据本仓库的 Makefile 与生成的镜像名可能不同，请以仓库实际 Makefile 为准。

## 开发说明

- 代码主要使用 C 语言实现关键逻辑，汇编用于启动、上下文切换与低级引导。
- 请先阅读 `kernel/`、`boot/` 和 `lib/kernel` 下的核心实现以理解启动流程与内核结构。
- 修改后可使用 `make` ；建议开启 GDB 的调试参数用于断点调试。

## 常见操作

- 编译：`make`
- 清理构建产物：`make clean`
- 运行（示例）：`make run` 或使用 `qemu-system-*` 手动运行生成镜像
